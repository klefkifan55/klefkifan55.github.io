5 Working with SQL Databases

Structured Query Language (SQL) is an essential tool for any data scientist working with large or complex datasets. In this chapter, I demonstrate how to load, clean, and integrate three biological datasets — dengue fever, influenza, and the Gapminder demographic dataset — into a PostgreSQL database, and use SQL queries to explore and join the data.
